# 🎙️ Voxa

> AI voices that sound natural.

Voxa is an open-source AI voice platform that converts text into natural-sounding speech using modern open-source text-to-speech models.

## ✨ Features (MVP)

- Convert text to speech
- Multiple AI voices
- Audio playback
- MP3 download
- Clean web interface

## 🛠️ Tech Stack

- Next.js
- FastAPI
- Python
- Tailwind CSS
- Kokoro TTS
